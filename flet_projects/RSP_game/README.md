### 📝 게임 규칙 및 조건
1. 사용자가 가위바위보 중 하나를 택한다./컴퓨터는 랜덤으로 선택된다.
2. 5판 3승제로, 총 5번의 게임을 진행하거나, 먼저 3승을 달성한 경우 게임이 종료된다.
3. 무승부 또한 전체 경기 횟수에 포함한다.
4. 게임 종료 후 승수가 높은 최종 승리자에게 문구가 표시된다.
